2. 测试是使用人工和自动手段来运行或检测某个系统的过程，其目的在于检验系统是否满足规定的需求，或者弄清预期结果与实际结果之间的差别
    测试的目的是不是为了证明程序是正确的，而是为了发现程序的缺陷，测试只能找出软件中的缺陷，不能证明软件没有缺陷，所以重点是如何保证尽量多的找出bug

3. 测试的几个基本原则
（1）尽早并不断的进行测试；
（2）测试用例应由"测试输入的数据"和对应的"预期输出结果"两部分组成；
（3）设计用例时，至少包含合理输入和不合理输入两种，边界值，等价交换类，边界条件，等等；
（4）全面检查每一次测试结果；
（5）保存好测试计划、测试用例、出错统计和最终分析报告，以便于维护。
